# Ray-Tracing
简单的光线追踪器

# 运行项目
mac/linux 环境下,控制台下运行 build.sh
windows 环境下,通过git的命令行工具运行 build.sh

mac 环境运行项目示例
```shell
cd desktop/ray-tracing #进入项目文件
chmod +x build.sh      #获得权限
./build.sh             #执行build.sh
```

# 效果说明
程序输出的是.ppm格式的图片
因为markdown不能展示.ppm,所以效果图都转成了.png格式
